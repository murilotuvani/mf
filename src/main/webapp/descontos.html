Definições

N = Valor nominal do título: importância a ser paga na data do vencimento.

A = Valor descontado (ou atualizado): valor pelo qual o tíutlo dev ser resgatado se for pago antecipadamente. Valor atual (VA).

Desconto

i = Vf-Va/Va

Desconto Racional Simples

Dm = A.in quando N = A (1 + in) => A = N / (1 + in)

Drs = N.in / (1 + in)

Desconto por fora ou desconto comerical

Dcs = Ndn

Drs = N.in / (1 + in)

Desconto racional composto

S = P x FSP => N = A x FPS

Ou A = N x FSP

Para definir FPS = N/A ou 1/(1+i)n

FPS = (1+i)n

FSP = 1/(1+i)n

Problemas dos descontos

  • Definir o desconto a paritr do valor nominal (N), período (n) e taxa (i)
  • Definir o valor atual de um título a partir do valor nominal (N), período (n) e taxa (i)
  • Definir o prazo (n) a partir do valor nominal (N), desconco, período (n) e taxa (i)
